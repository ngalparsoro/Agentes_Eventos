# Repositorio historico - Agentes_Eventos

Este repositorio queda como historico/desarrollo previo del equipo de Data Science.

El repositorio oficial y canonico para el backend DS desplegado en Render es:

```text
https://github.com/ngalparsoro/despliegue_agentes
```

URL de produccion:

```text
https://despliegue-agentes.onrender.com
```

A partir de ahora, cualquier cambio operativo que deba consumir Front/FS o que deba llegar a Render debe hacerse en `despliegue_agentes`.

Este repo puede conservarse para consultar documentacion antigua, pruebas o material de desarrollo, pero no debe usarse como fuente de verdad para integracion.
